# BNET Smart Monitoring

BNET Smart Monitoring adalah aplikasi sederhana untuk melakukan analisa kondisi jaringan berdasarkan beberapa parameter seperti RX Power, Traffic Client, Latency, dan Packet Loss.

## Fitur
- Analisa RX Power
- Monitoring Traffic Client
- Analisa Latency
- Deteksi Packet Loss
- Mendapatkan recomendation action agar bisa mengatasi trouble tersebut
- Riwayat monitoring

## Cara Menggunakan
1. Masukkan nilai RX Power
2. Masukkan nilai Traffic Client
3. Masukkan nilai Latency
4. Masukkan nilai Packet Loss
5. Klik tombol "Analisa Jaringan" untuk mendapatkan hasil analisa jaringan tersebut

## Teknologi yang di gunakan 
- HTML
- CSS
- JavaScript
